# Exploratory Data Analysis (EDA)

# Project Overview
This project performs Exploratory Data Analysis (EDA) on a cleaned sales dataset to uncover patterns, trends, and business insights. Using Python and data visualization libraries, the analysis helps understand sales performance, customer behavior, product demand, and revenue trends.

# Objectives
* Explore and analyze sales data.
* Identify trends and patterns.
* Detect outliers and anomalies.
* Visualize key business metrics.
* Generate actionable insights.
